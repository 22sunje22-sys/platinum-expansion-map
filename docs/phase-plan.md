# Phase plan

Moat-led Phase 1 · execution plays bundled · licensing for no-moat markets.

## Portfolio logic

Phase 1 must include **one summer market + one shoulder market** to smooth revenue seasonality:
- **Summer:** Italy (peak Jun–Sep), Malaysia (Jul–Aug Khaleeji season)
- **Shoulder:** Spain (Mar–May, Oct–Nov), Singapore (year-round family-tech), Jordan (Mar–May, Oct–Nov)

Moat leaders (high GCC bias ratio) anchor the phase; execution plays (high supply + iconic brand but thin moat) run off the back of the moat-led launches to share operational infrastructure.
